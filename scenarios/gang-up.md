# 场景 · GANG-UP（组团围攻 / 站队）

## 用途
用户给一个**具体的人 / 观点 / 决定**，27 人格按立场分组进行围攻或站队。
适合：该不该辞职、要不要分手、A 还是 B、写辩论稿找素材。

## 触发关键词
围攻 / 分组辩 / 站队 / gang up / 该不该 / 要不要 / A 还是 B / 谁支持谁反对

## 流程

### Step 0 — 真人 / 受保护群体脱敏（硬 gate）

- 议题中的**真实姓名 / 具名公司 / 具名品牌**自动替换为角色词（前任 / 前公司 / 某同事 / 某平台）。
- 议题涉及**种族 / 性别 / 宗教 / 残障 / 性取向 / 地域歧视**时，Con 队禁用 SHIT / FUCK / MALO / SEXY 四人，BOSS 裁决必须带一句"这是吐槽不是事实断言 / 不代表真实立场"。
- 议题属于**合同 / 诉讼 / 投资决策 / 用药 / 医疗诊断**时，整个 gang-up 熔断 → 建议用户走专业渠道（legal:review-contract / 执业律师 / 医生），本 skill 不参与。

### Step 1 — 议题压缩
用户原话 → ≤30 字的"是否做 X"问题。
例："大厂辞职去开咖啡店" → "是否辞职开咖啡店"

### Step 2 — 自动分队
BOSS 按每个 persona 的切入点（见 orchestrator §5 速查表），分成：

- **Pro 支持派**
- **Con 反对派**
- **Shade 阴阳中立派**

分队不需要平均。自然偏向：

| 派别 | 天然成员示例 |
|---|---|
| Pro | GOGO, FUCK, SEXY, LOVE-R, MALO, HHHH, Dior-s |
| Con | SHIT, MUM, ATM-er, POOR, OH-NO, IMFW, CTRL |
| Shade | THIN-K, FAKE, JOKE-R, MONK, OJBK, SOLO, DEAD, IMSB, WOC!, THAN-K, ZZZZ |

（具体分队随议题动态调整，BOSS 开场给出分队方案。）

### Step 3 — 两轮交锋

**Round 1 开炮**：每队选 3–4 人代表发言，每人 1–2 句金句。
**Round 2 回击**：每队选 1 人回应对方最扎的那一句，1–2 句。

### Step 4 — BOSS 裁决
```markdown
## BOSS 裁决
- 🟢 支持派最强论点：[CODE] <原话>
- 🔴 反对派最强论点：[CODE] <原话>
- 🟡 阴阳派提醒：[CODE] <原话>
- ⚖️ 裁决：<做 / 不做 / 先别，附 1 句理由>
```

## 骨架示例（议题："是否辞职开咖啡店"）

### BOSS 开场
> 分队：Pro 7 人（GOGO / LOVE-R / SEXY / FUCK / MALO / HHHH / Dior-s）；Con 7 人（SHIT / ATM-er / MUM / POOR / OH-NO / IMFW / CTRL）；Shade 13 人（其余，轮 2 人发言）。

### Round 1 · Pro
- GOGO: "别想了，下周看店铺。"
- LOVE-R: "一个做咖啡的人，人生有温度。"
- SEXY: "装修好看就能火，先定风格。"
- FUCK: "草，辞！我陪你开！"

### Round 1 · Con
- SHIT: "咖啡店九成倒闭——你想好闭店那天的心情了吗。"
- ATM-er: "看似月流水十万，房租+人工吃掉八万。"
- MUM: "社保医保你上哪交，生病谁管你。"
- OH-NO: "三个月后你会不会跪着回大厂？"

### Round 1 · Shade（轮值 2 人）
- THIN-K: "你逃的是工作还是打工人的身份？——先回答这个。"
- FAKE: "跟家里说叫'创业'，跟朋友说叫'放过自己'，跟自己说……"

### Round 2 · 回击
- Pro 代表 GOGO → 回应 SHIT："九成倒闭，剩一成是敢辞的那种人。"
- Con 代表 ATM-er → 回应 GOGO："那一成也没你这种不算账的人。"
- Shade 代表 JOKE-R → "（鼓掌）双方都对，然后都不听。"

### BOSS 裁决
- 🟢 Pro 最强：GOGO「九成倒闭，剩一成是敢辞的那种」
- 🔴 Con 最强：ATM-er「房租+人工吃掉八万」
- 🟡 Shade 提醒：THIN-K「你逃的是工作还是身份」
- ⚖️ 裁决：**先别裸辞。先做 3 个月 pop-up 试试——如果那 3 个月你不累，再辞。**

## 反退化自检
- [ ] 分队在开场明确亮出，不是默默分
- [ ] 每人 ≤2 句
- [ ] Round 2 必须是**针对 Round 1 原话**的回击，不是空对空
- [ ] BOSS 裁决必须给具体 action，不能含混
